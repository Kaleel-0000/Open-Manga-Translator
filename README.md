# Manga Translate 🌐📖

Open-source browser extension that translates manga, comics, manhwa, manhua, 
and webtoons directly in your browser — no external apps, no subscriptions, 
no watermarks.

Translated text is rendered back into the original speech bubbles with the 
artwork fully preserved, so pages look like they were originally published 
in your language.

## 100% Free & Open Source
Licensed under MIT. Use it, fork it, improve it, build on it.
No paywalls. No premium tier. No data collection.
Unlimited translations when using local providers like Ollama or LM Studio.

## Supported Browsers
Chrome • Edge • Brave • Opera • Firefox

## Supported Content
Manga • Manhwa • Manhua • Webtoons • Comics • Infinite scroll readers • 
Lazy-loaded image sites

## Translation Providers
Local (offline/free): Ollama, LM Studio, LibreTranslate
Cloud: OpenAI, Claude, DeepL, Google Translate

## OCR Providers
Tesseract.js (offline) • Google Cloud Vision • Azure Computer Vision

## Key Features
- Auto-detects comic images on any page
- Removes original text with smart inpainting (preserves artwork)
- Bubble-aware text rendering with auto font sizing
- Toggle between original and translated view
- Full offline mode with local providers
- IndexedDB caching — revisiting pages is instant
- No limits imposed by the extension itself
